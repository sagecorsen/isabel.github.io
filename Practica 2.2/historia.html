Web 3.0

"La Web Semántica"

Introducción de HTML5 como estándar para aplicaciones web, impulsando la estandarización y mejorando significativamente la experiencia del usuario gracias a funciones avanzadas, mayor interactividad y compatibilidad multiplataforma.

Se comienza a popularizar el internet de las cosas, abriendo la oportunidad de que todo esté conectado a través de la red , como electrodomésticos inteligentes, dispositivos de seguridad, sistemas de iluminación y transporte , mejorando la eficiencia, la comodidad y el control en nuestra vida cotidiana.

Web 4.0

"La Web Obiqua"

Donde la web no solo mostraría información, sino que además interactuaría de forma proactiva con el usuario.

Fue en este periodo cuando algunos expertos comenzaron a hablar de la "web activa" y contextual, sentando las bases conceptuales para lo que vendría.
